# ローカルサーバー起動手順

## 起動コマンド

ターミナルで以下を実行：

```bash
cd "/Users/seira/Documents/claudecode/260408_yuレクチャー/聖来のウェブサイト"
npx browser-sync start --server --files "**/*"
```

## アクセス方法

| 端末 | URL |
|------|-----|
| PC | http://localhost:3000 |
| スマホ（同じWi-Fi） | http://192.168.100.38:3000 |

※ スマホのURLはWi-Fi環境によって変わる場合あり。その場合は以下で確認：
```bash
ipconfig getifaddr en0
```

## 特徴

- ファイルを保存すると**PC・スマホ両方が自動リロード**される
- 終了するときは `Ctrl + C`
